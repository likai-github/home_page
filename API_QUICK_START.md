# 🚀 API 管理快速开始

## 5 分钟上手新的 API 管理系统

---

## 第 1 步：重置数据库（如果需要）

如果你之前使用过旧版本，需要先重置数据库：

```
访问：https://gaoying.qzz.io/api/reset-db?confirm=yes
```

然后执行迁移：

```
访问：https://gaoying.qzz.io/api/migrate
```

---

## 第 2 步：添加 NVIDIA 平台

1. 登录管理后台：https://gaoying.qzz.io
2. 进入 **API 管理** 页面
3. 点击 **➕ 添加平台**
4. 填写信息：
   - **平台类型**: 选择 `nvidia`
   - **显示名称**: `NVIDIA API`
   - **API Key**: `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`
   - **Base URL**: 留空（使用默认）
   - **描述**: `NVIDIA AI 模型平台`
   - **状态**: `启用`
5. 点击 **保存**

---

## 第 3 步：同步模型列表

1. 在 NVIDIA 平台卡片上点击 **📋** 图标
2. 点击 **🔄 同步模型** 按钮
3. 等待同步完成（约 5-10 秒）
4. 看到成功提示：`✓ Models synced successfully (新增: 25, 更新: 0)`

---

## 第 4 步：启用需要的模型

在模型列表中，找到你需要的模型并启用：

**推荐模型：**
- ✅ `meta/llama-3.1-8b-instruct` - 快速、高效
- ✅ `nvidia/nemotron-4-340b-instruct` - 强大、准确
- ✅ `mistralai/mixtral-8x7b-instruct-v0.1` - 平衡性能

使用右侧的开关切换模型状态。

---

## 第 5 步：（可选）添加其他平台

### 添加 OpenAI

1. 点击 **➕ 添加平台**
2. 填写：
   - **平台类型**: `openai`
   - **显示名称**: `OpenAI API`
   - **API Key**: 你的 OpenAI API Key
   - **Base URL**: 留空
3. 保存并同步模型

### 添加 Anthropic

1. 点击 **➕ 添加平台**
2. 填写：
   - **平台类型**: `anthropic`
   - **显示名称**: `Anthropic API`
   - **API Key**: 你的 Anthropic API Key
   - **Base URL**: 留空
3. 保存并同步模型

---

## ✅ 完成！

现在你已经成功配置了 API 管理系统！

- 📊 查看所有平台的统计信息

- 🎛️ 灵活启用/禁用模型
- ✏️ 编辑平台配置
