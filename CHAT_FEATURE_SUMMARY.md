## 🔧 技术实现

### 模型选择

```javascript
// 从 API 平台管理系统加载已启用的模型
const loadAvailableModels = async () => {
  const data = await api.getPlatforms();
  const platforms = data.platforms || [];
  
  // 为每个平台加载已启用的模型
  for (const platform of platforms) {
    if (platform.status === 'active') {
      const modelsData = await api.getPlatformModels(platform.id);
      platform.models = modelsData.models.filter(m => m.enabled);
    }
  }
  
  availablePlatforms.value = platforms.filter(
    p => p.models && p.models.length > 0
  );
};
```

### 对话历史存储

// 保存到 localStorage

## 🎯 使用流程

### 首次使用

1. **访问聊天页面**
   - 从首页点击"开始对话"
   - 或从导航栏点击"AI 对话"

2. **选择模型**
   - 在下拉框中选择一个 AI 模型
   - 模型按平台分组显示

3. **开始对话**
   - 点击示例提示快速开始
   - 或在输入框输入自己的问题

### 继续对话

1. **查看历史**
   - 侧边栏显示所有对话
   - 按时间分组（今天/更早）

2. **切换对话**
   - 点击历史记录切换
   - 消息自动加载

3. **删除对话**
   - 悬停在对话上显示删除按钮
   - 点击确认删除
